# Fifth Echo

A dependency-free playable prototype of a party-building card roguelite across five realities.

## Play online

https://noodels91hub.github.io/fifth-echo/

## Play locally

Open `index.html` directly in a browser, or run any simple static file server in this directory.

For example:

```powershell
python -m http.server 4173
```

Then visit `http://localhost:4173`.
